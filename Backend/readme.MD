# Instructions

## Specification
`npm 7.5.x`
`nodejs 12.22.x`
`Pop!_Os 21.10`

## Initiaize

``` node
npm init
```

## Modules
``` node
npm i express bcryptjs jsonwebtoken mongoose express-validator --save
```

## Nodemon global setup

``` bash
sudo npm install -g nodemon
```

## Start Running program
``` node
nodemon
```
